## tplus-cli
##### tplus-cli是帮助你快速创建生成项目的脚手架工具，配置了多种可选择的不同类型项目模版。

### Installation

```
npm install tplus-cli -g
```
### command

#### 生成项目
```
tplus-cli init
```

| 操作  | 默认  |
| ------------ |
|  输入项目名称 |tplus-project|
|  选择创建模板 ||
|  输入项目版本 |1.0.0|
|  输入项目描述 |tplus project|
|  输入项目作者 |tplus|
|  是否自动安装依赖 |y |

#### 更新
```
tplus-cli update
```
